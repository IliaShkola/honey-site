How It Works

1. Launch from Folder

Right-click on any empty space in a folder containing IFC files and select “Open with HoneyIFC”. The app will automatically list all IFC files in that folder.

2. Browse & Select

Navigate through categories and property sets. Pick only what you need – categories and psets.

3. Export Clean Data

Get a focused Excel file with exactly your selected data. No cleanup required.
